Mos Fit is an AI-driven diet tracking web app built using the **React framework**.

The application utilizes the React framework for web-app front-end development. It collects data by utilizing public APIs and developing internal APIs for its back-end systems.

**Technologies Used:**
* TypeScript
* React
* MySQL
* Git
* API
