**Role:** Software Engineer

Developed complex backend systems for environmental monitoring and AI integration.

* **Backend:** Built a Flask-based backend integrating AI for real-time speech-to-text and environmental audio classification.
* **Offline Capabilities:** Developed a custom offline map-tile server with a multi-threaded downloader (`concurrent.futures`), enabling full operation without internet connectivity.
* **AI Integration:** Created an "AI Advisor" module that interfaces with local LLMs and generates automated PDF reports using FPDF.
* **Frontend:** Designed a responsive, themeable frontend using vanilla JavaScript and Leaflet.js for real-time sensor tracking and map visualization.
